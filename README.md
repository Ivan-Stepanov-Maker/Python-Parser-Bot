# Python-Parser-Bot

Данный репозиторий содержит готовый код на python телеграмм чат-бота.

## Что он умеет

- 1.Безопасно парсить товары с WB и Ozon по разным категориям.
- 2.Предоставлять данные о товарах(артикуль, цена и количество на складе) в реальном времени.
- 3.Отслеживать трафик в админ-панели.

## Как пользоваться

#### 1.Подготовка окружения

- 1.Установите новейшую версию python на [официальльном сайте](https://www.python.org/downloads/).
- 2.Скачайте папку и откройте её в любой IDE(VS code/PyCharm).
- 3.Откройте терминал и введите: *pip install aiogram, requests, BeautifulSoup4*(Если pip не добавлен в bin, тогда введите: *Python -m pip install aiogram, requests, BeautifulSoup4.*
- 4.Напишите в терминале: *myenv\Scripts\Activate*.

#### 2.Подключения бота и запуск

- 1.Зайдите в телеграмм и найдите @BotFather.
- 2.Создайте бота, дав ему уникальное имя.
- 3.Получите уникальный токен(**НИКОМУ НЕ СООБЩАЙТЕ ЕГО! ЭТО ДАЁТ ДОСТУП К БОТУ**).
- 4.Зайдите обратно в IDE и добавьте в основной папке файл: **config.py**(**Данный файл содердит констатные переменные, которые не должны быть доступны пользователю, но и без них бот будет простым куском железа**).
- 5.В этом файле создайте констатную переменную TOKEN и впишите туда токен, который получили от @BotFather(Пример: *TOKEN='ВАШ ТОКЕН'*).
- 6.После переменной TOKEN снизу добавьте переменную ADMIN_ID - туда нужно вставить ваш ID профиля, чтобы быть администратором(Пример: *ADMIN_ID=ВАШ ID(БЕЗ КАВЫЧЕК, ТАК КАК ЭТО ЧИСЛО, А НЕ СТРОКА*).
- 7.Сохрание изменения через **CTRL + S** и в терминале напишите: *Python run.py*, после чего попробуйте написать /start, если бот нечего ненапишет в течении 5 секунд, значит это возможно проблемы с интернетом или нету доступа к Telegram API, такое бывает у жителей в РФ.
